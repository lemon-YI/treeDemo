### 使用步骤（这里以在vue-cli3的项目为例）
```

Ⅰ.安装：这边安装一定要加-D
npm install -D less-loader less
npm install -D vue2-org-tree

Ⅱ.在main.js引入注册
import Vue2OrgTree from 'vue2-org-tree';
Vue.use(Vue2OrgTree)

```
